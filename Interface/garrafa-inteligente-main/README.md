# garrafa_inteligente

Projeto de um aplicativo multiplataforma para ser utilizado no projeto 'Garrafa Inteligente'.

## Desenvolvimento
O propósito do aplicativo é mostrar as informações auais e o histórico coletdo pela garrafa térmica em um aplicativo que possui o funcionameno de sincronicidade de telas entre todas suas versões, para que a experiência seja contínua
